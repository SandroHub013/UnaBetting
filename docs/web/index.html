The leak hunt

Every inflated number, hunted down.

This project's real story is a chain of self-caught data leaks. Each one made the model look brilliant. Each one was a bug.

2026-05-21

Pre-split imputation

Medians computed over the full dataset leaked the future into training.

acc 78.8% → train-only medians
2026-05-22

Unpaired perspective columns

A w_ feature with no l_ twin reconstructed the target through the 50% flip.

ROI +46% → perspective pairs enforced
2026-06-09

The fake 85% backtest

No scaler, un-randomized rows, phantom even-money odds. The bankroll "exploded" — on a bug.

win rate 85% → −92% honest ROI
